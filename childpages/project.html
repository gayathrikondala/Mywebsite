IOT Based Automatic Saline Monitoring System using Node MCU

An IOT-based Automatic Saline Monitoring System is proposed to monitor the level of electrolyte. When fluid level is low, it will alert the observer through an alarm at the control room.

Generally, in almost all hospitals, an assistant or nurse is responsible for monitoring the level of the electrolyte. Unfortunately, sometimes they may forget to change the bottle at the right time due to their busy schedule or unexpected circumstances. This may create a problem for the patient because the reverse flow of blood will start, potentially leading to severe infections.

To overcome this situation, an IoT-based Automatic Saline Monitoring System is proposed to monitor the level of electrolyte. Frequently the quantity of saline is measured, and the data is stored in the cloud. By using the data, we can alert the observer through the app and alarm, as well as sending a text message to the doctorâ€™s mobile phone (similarly to how we are receiving notification of mobile data usage.

Advantages
  • Continuous monitoring of saline levels.
  • Reduces the risk of reverse blood flow and infections.
  • Automated alerts to nurses and doctors.
  • Data storage in the cloud for future reference.

Labeling Of The Images In The DataSets Using Deep Learning Alogorithms

Trained a neural network model to classify images of clothing like sneakers and shirts.

In this technique, a model is trained on a large dataset of labeled images, where each image is associated with one or more predefined labels. The model learns the patterns and features that are indicative of each label, allowing it to predict the appropriate labels for new, unseen images.

Image labeling is a technique for recognizing and labeling various elements in images. These tags are then used to train computer vision models to recognize specific objects in images.Data labeling in computer vision is the process of applying tags to raw data such as images and videos. Each tag specifies an object class connected with the data, which includes photographs and videos. Each tag indicates an object class connected with the data. Unstructured data is labeled by supervised machine learning algorithms to detect a given item type. It allows these models to make sense of the data, which benefits model development.

Advantages
  • Increased accuracy in predictions.
  • Automated feature extraction.
  • Ability to handle large amounts of data.
  • Improved decision-making processes.
